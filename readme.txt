== Description  ==

Pulsante per i blogger che vogliono proporre su Upnews.it o far votare i propri contenuti direttamente dal loro Blog. Verranno visualizzati il numero di voti ed il pulsante per votare. Nel caso la news non sia ancora presente nel sistema, sul vosto blog appare il pulsante proponi.

[Plugin Homepage](http://www.upnews.it/tools/ "Plugin Homepage")

== Installation ==

1. Scarica il plugin per wordpress.
2. Inserisci il file upnews-plugin.php all'interno della cartella plugins.
4. Attivalo e configuralo.
